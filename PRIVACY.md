# Bibilili Privacy Policy

Effective date: April 26, 2026

Bibilili is a Chrome extension for Bilibili watch pages. It rearranges the Bilibili player, comments, and video lists into a transformed viewing layout.

## Data handled

Bibilili reads content from supported Bilibili pages, including player, comment, title, and video-list elements needed to render the transformed layout.

When the user is signed in to Bilibili, Bibilili may request the user's Bilibili watch-later and history video lists from `https://api.bilibili.com/*`. These requests are made directly from the browser to Bilibili using the user's existing Bilibili session.

Bibilili stores a local enabled or disabled preference in the browser so the layout state can persist across page reloads.

## How data is used

Bibilili uses Bilibili page content and Bilibili API responses only to provide the extension's watch-page layout, source buttons, and bottom video dock.

## Data sharing

Bibilili does not send user data to the developer, analytics services, advertising services, or any non-Bilibili server.

Bibilili communicates with Bilibili APIs only to fetch Bilibili account video-list data used by the extension.

## Data retention

Bibilili does not retain user data on developer-controlled servers.

The local enabled or disabled preference remains in the browser until the user changes it, clears site or extension data, or uninstalls the extension.

## Remote code

Bibilili does not load or execute remote JavaScript or WebAssembly. All extension code is included in the extension package.
